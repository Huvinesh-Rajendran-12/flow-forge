## HR Manager
- **Responsibilities:** Hiring decisions, onboarding coordination, policy enforcement, benefits administration
- **System access:** HR Portal (admin), Google Workspace (admin), Slack (member)
- **Onboarding duties:** Initiate onboarding workflow, create employee record, assign buddy, verify compliance docs

## IT Admin
- **Responsibilities:** Account provisioning, access management, equipment setup, security compliance
- **System access:** All systems (admin)
- **Onboarding duties:** Create accounts across all systems, configure hardware, set up VPN/SSO credentials, grant repo access

## Team Lead
- **Responsibilities:** Sprint planning, task assignment, team introductions, performance reviews
- **System access:** Jira (project admin), GitHub (team maintainer), Slack (channel admin)
- **Onboarding duties:** Create onboarding Jira tasks, introduce new hire to team, set initial performance goals, schedule 1:1s

## New Employee
- **Responsibilities:** Complete onboarding checklist, attend training, set up workstation
- **System access:** Granted progressively during onboarding
- **Onboarding duties:** Accept invitations, complete compliance training, enroll in benefits, set up dev environment
